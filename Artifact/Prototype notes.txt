GENERATING PACKAGE LIST:
    pip list --format=freeze > requirements.txt

Using pip list with format rather than directly calling pip freeze because it's a conda-pip hybrid environment, where pip 
freeze generates conda URI links rather than direct package versions, making the requirements.txt file unusable by pip.
